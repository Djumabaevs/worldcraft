# 🏗️ WorldCraft — 3D город из твоих привычек

> Каждая привычка — здание. Каждый день — новый этаж. Пропустил — обвал. Построй свой город.

---

## 📖 Концепция

WorldCraft — это **трекер привычек, где прогресс = 3D город**. Вместо скучных галочек ты строишь красивый изометрический город. Каждая привычка — здание определённого типа. Выполняешь привычку каждый день — здание растёт. Пропускаешь — здание разрушается. Цель: построить мегаполис.

**Почему работает:** Геймификация + визуализация прогресса + "жалко разрушать" = сильнейшая мотивация.

## 🏢 Система зданий

### Типы зданий по привычкам
```
💧 Вода      → Фонтан → Водонапорная башня → Аквапарк
💪 Спорт     → Площадка → Спортзал → Стадион
📚 Чтение    → Скамейка → Библиотека → Университет
🧘 Медитация → Камень → Храм → Пагода
🏃 Бег       → Дорожка → Парк → Стадион бега
😴 Сон       → Палатка → Дом → Отель
📝 Учёба     → Парта → Школа → Академия
🎨 Творчество→ Мольберт → Студия → Галерея
🍎 Питание   → Грядка → Кафе → Ресторан
```

### Уровни зданий (streak дней)
```
Streak 0:     Пустой участок (развалины если было здание)
Streak 1-3:   Уровень 1 — маленькое (1 этаж)
Streak 4-7:   Уровень 2 — среднее (2-3 этажа)
Streak 8-14:  Уровень 3 — большое (4-5 этажей)
Streak 15-30: Уровень 4 — впечатляющее (6-8 этажей + декор)
Streak 31-60: Уровень 5 — монументальное (9-12 этажей + свет + частицы)
Streak 61+:   Уровень 6 — легендарное (небоскрёб + аура + анимации)
```

### Разрушение (пропуск дня)
```
Пропуск 1 день:  Здание темнеет, трещины
Пропуск 2 дня:   -1 уровень, анимация обвала
Пропуск 3+ дней: Здание = развалины, streak = 0
```

### Декорации (автоматические, от общего прогресса)
```
XP 0-100:     Пустая трава
XP 100-300:   Дороги между зданиями
XP 300-500:   Деревья, кусты
XP 500-1000:  Парк, скамейки, фонари
XP 1000-2000: Река, мосты
XP 2000-5000: Машины на дорогах, люди (точки)
XP 5000+:     Метро, аэропорт, небоскрёбы фона
```

## 🔧 Техническая реализация

### 3D город (Three.js, изометрическая камера)
```javascript
// Изометрическая камера
const camera = new THREE.OrthographicCamera(
  -15, 15, 15, -15, 0.1, 1000
);
camera.position.set(20, 20, 20);
camera.lookAt(0, 0, 0);

// Сетка города: 8x8 ячеек
// Каждая ячейка = 2x2 единицы
// Привычки размещаются по сетке
// Дороги между ними
// Декорации в свободных ячейках

// Здания: процедурные из BoxGeometry
// Каждый уровень = +1 этаж (BoxGeometry stack)
// Цвет = категория привычки
// Окна = маленькие emissive плоскости
// Крыша = пирамида или плоская

function buildBuilding(habit, streak) {
  const group = new THREE.Group();
  const floors = getFloorsForStreak(streak);
  const color = CATEGORY_COLORS[habit.category];

  for (let i = 0; i < floors; i++) {
    const floor = new THREE.Mesh(
      new THREE.BoxGeometry(1.5, 0.8, 1.5),
      new THREE.MeshStandardMaterial({ color })
    );
    floor.position.y = i * 0.8 + 0.4;
    group.add(floor);

    // Окна
    if (i > 0) {
      addWindows(group, i, color);
    }
  }

  // Крыша
  addRoof(group, floors, habit.category);

  return group;
}
```

## 📱 Адаптивность

### Desktop (>1024px)
- 3D город: 70% экрана (слева)
- Чеклист + статы: 30% (справа, панель)
- Drag + scroll для камеры

### Tablet (768-1024px)
- 3D город: верхняя часть (55%)
- Чеклист: нижняя часть (45%, scrollable)

### Mobile (<768px)
- 3D город: верх (40%, компактный)
- Чеклист: низ (60%)
- Swipe tabs: Сегодня / Город / Стата
- Упрощённые здания (меньше деталей)
